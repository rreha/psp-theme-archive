Installation

Put the .ptf files to your PSP's theme directory "X:/PSP/THEME/".

Some of the themes are in .7z/.zip format, those have alternative wallpapers inside. Extract them using 7-Zip/WinRAR or any unarchive program of your choice. Put the images inside the "Alternative Wallpapers" folder to "X:/PSP/PHOTO/" or "X:/PICTURE/" (X: is your PSP's drive letter).

Please report me if there are any broken/missing links!

Also please note that some themes might be NSFW.

Enjoy!!
